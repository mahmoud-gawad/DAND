# Explore US Bikeshare Data

This is the first project using Python for data analysis. It is required to look at data about bikesharing system from different cities, clean them, wrangle them, and draw insights from them using principles of statistics and grahs.

## Full Report

The full report for this project can be found inside the IPYNP file, or the HTML file in case you don't have Jupyter installed.

## Key Takeaways

- Introduction to Jupyter Notebooks and learning how to use them.
- Introduction to data wrangling.
- Learning how to visualize different types of charts.
- Data analysis skills like asking data-related questions and making sense of the present data.
